Laboratorio 1

Familiarize the student with: variables (i.e. naming, declaring, initializing and modifying their values)

Laboratorio 2

Familiarize the student with: the basic properties of complex data types such as Array and Object (treated as a record) and be able to use them in practice.

Laboratorio 3

Interacting with the user (dialog boxes – alert, confirm, prompt)

Laboratorio 4

Conditional execution (what is conditional execution, the if–else statement, the conditional operator, the switch–case statement)

Laboratorio 5

loops (what are loops, the while loop, the do–while loop, the for loop, the for–of loop, the for–in loop, the break and continue statements)

Laboratorio 6

Familiarize the student with: function basics, functions as first-class members, arrow functions, recursion.
